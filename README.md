# Python-GUI-Calculator---Basic

A Basic Calculator with Graphical User Interface made using Python tkinter.

tkinter is a python built-in library for creating GUI applications.

This is a simple calculator with buttons for ADDITION, SUBTRACTION, MULTIPLICATION, DIVISION and MODULUS. 

This calculators design was inteded to be similar to the built-in windows calculator.

It is a simple application i made while getting myself familiar with the Python GUI tkinter library.
